# 0. 농도

농도에 대해 다시 한번 기억하고 넘어갈 필요성이 존재할 확률이 높다.

## 1. 질량을 이용한 농도

$$퍼센트 농도 (\%) = \frac{용질의 질량(g)}{용액의 질량(g)} \times {100}$$

## 2. 몰 농도 ($M, mol/L$)

$$M = \frac{n (mol)}{V (L)}$$ -> $$n = MV$$

### 기체의 경우

기체 1L 속에 있는 기체의 몰수  
$$M = \frac{기체의 몰수 (mol)}{기체(용기)의 부피 (L)}$$

### 수용액의 경우

수용액 1L 속에 녹아 있는 용질의 몰수  
$$M = \frac{용질의 몰수 (mol)}{용액의 부피 (L)}$$

### 고체 or 액체일 경우

고체 or 액체 1L 속에 있는 고체 or 액체의 몰수

### 농도의 표시

$[A]$ = A의 농도를 나타내고, $[A]_0$ = A의 초기 농도를 나타낸다.

# 1. $N_2O_4-NO_2$ Equilibrium System

## 화학 평형

### 평형상태: **정반응** 속도와 **역반응** 속도가 같은 동적 평형 상태

$$N_2O_4(g) \rightleftarrows 2NO_2(g)$$

- 화학 평형은 가역 반응에서만 이루어질 수 있다.
- 반응 물질과 생성 물질이 모두 존재하며 반응물과 생성물의 농도는 일정하게 유지된다.
- 화학 반응식의 계수비는 평형 상태의 물질의 몰 농도 비와 무관하다. 계수비는 초기 상태에서 평형에 도달하기까지 반응 물질이 감소한 농도와 생성 물질이 증가한 몰 농도의 비를 의미한다.

정반응 속도와 역반응 속도는 평형을 이룸에 따라 서로 같아진다.

# 2. Equilibrium Constant Expression

## 1. 평형 상수 $K_p$ (기체상 반응)

$$aA(g) + bB(g) \rightleftarrows cC(g)$$  
$$K_p = \frac{P_C^c}{P_A^a \cdot P_B^b}$$  
($P_A, P_B, P_C$)는 각각 평형 상태에서 각 물질의 분압이다.

- 평형 상수: 화학 반응식의 계수를 분압의 지수로 한 생성 물질의 분압 곱을 반응 물질의 분압 곱으로 나눈 값
- 단위: 생략
- 평형 상수는 **온도**에만 의존한다
- 평형 위치: 평형 상태에서 각 경우의 평형 분압

## 2. 농도 평형 상수 $K_c$

$$aA(g) + bB(g) \rightleftarrows cC(g)$$  
$$K_c = \frac{[C]^c}{[A]^a[B]^b}$$  
($[A], [B], [C]$)는 각각 평형 상태에서 각 물질의 몰 농도이다.

여기서 $K_c$와 $K_p$의 관계식을 유도해낼 수가 있는데,

$$K_p = \frac{P_C^c}{P_A^a \cdot P_B^b} = \frac{(\frac{n_CRT}{V})}{(\frac{n_ART}{V})^a(\frac{n_BRT}{V})^b} = \frac{[C]^c}{[A]^a[B]^b}\times (RT)^{c-(a+b)}$$

정리하면
$$K_p = K_c(RT)^{c-(a+b)}$$이고,

$$c-(a+b)$$는
$$\Delta n_g$$라고 해서, 기체의 몰수 변화라 할 수 있다.

- 온도가 일정하면 $$K_c, K_p$$ 모두 일정하다.

여기서 외울건

$$K_p = \frac{P_C^c}{P_A^a \cdot P_B^b}$$

$$K_c = \frac{[C]^c}{[A]^a[B]^b}$$

$$K_p = K_c(RT)^{\Delta n_g}$$

이 3가지가 되겠다.

## 3. Heterogeneous 평형에서의 평형상수식

만약 (g) 상태의 물질들로만 이루어진 것이 아닌, (s), (l), (g), (aq) 등이 다 같이 들어있는
반응식이라 해보자.  
(g)는 부분압으로 atm단위로 나타내고, (aq)는 몰 농도로 나타내서 계산해내면 된다.
(l)과 (s)는 1로 취급한다. (사실 상 무시)  
예시를 들자면,
$$2H_2O(l) \rightleftarrows 2H_2(g) + O_2(g)$$ 에서

$$K = \frac{P_{H2}^2P_{O_2}}{1}$$

## 4. 평형 상수의 성질

- 역반응의 평형상수는 정반응의 평형 상수와 역수 관계이다.
- 화학반응식을 n배하면 이떄의 평형 상수는 $$K^n$$ 의 관계가 있다.
- 첫 번 째 화학 반응식의 평형 상수를 $$K_1$$이라 하고 두 번째 화학 반응식의 평형 상수를 $$K_2$$라고 하고,
  두 화학 반응식을 더하여 나온 화학 반응식의 평형 상수는 $$K_1 \times K_2$$이다.
- 만양 두 화학 반응식을 뺐다면 $$\frac{K_1}{K_2}$$가 된다.

# 3. Applications of the Equilibrium Constant

## 1. Direction of Reaction: the Reaction Quotient(Q)

- 반응 지수 Q: 현재 상태에서의 반응물과 생성물의 분압비

$$aA(g) + bB(g) \rightleftarrows cC(g)$$

$$Q = \frac{P_C^c}{P_A^a \cdot P_B^b}$$

($P_A, P_B, P_C$)는 각각 **현재** 상태에서 각 물질의 분압이다.

- $$Q < K$$: 정반응 쪽으로 반응 진행
- $$Q = K$$: 평형 상태
- $$Q > K$$: 역반응 쪽으로 반응 진행

## 2. 평형 상수 값의 의미

- 평형 상수 값이 클 때: 반응이 정반응 쪽으로 우세하게 진행되어 평형 상태에서 생성 물질의 양이 반응 물질의 양보다 크다.
- 평형 상수 값이 작을 때: 반응이 역반응쪽으로 우세하게 진행되어 평형 상태에서 반응 물질의 양이 생성 물질의 양보다 크다.

평형 상수와 반응 속도는 관계가 없다.
반응 속도는 활성화 에너지의 크기에 의해 결정되는 것이다.

## 3. 평형 문제 접근

- K하고 Q로 정반응인지 역반응인지 파악한다.
- ICE를 작성한다
- 계산한다.

한 문제를 예시로 들어보자면,

> For the System  
> $$CO_2(g) + H_2(g) \rightleftarrows CO(g) + H_2O(g)$$  
> K is 0.64 at 900 K.  
> Calculate the equilibrium partial pressures of all species, starting with
> $$P_{CO_2} = P_{H_2} = 1.00atm; P_{CO} = P_{H_2O} = 0$$

여기서 먼저 Q를 계산해준다.

$$Q = \frac{0 \times 0}{1 \times 1} = 0$$

Q = 0이고, K = 0.64 이므로,
$$K > Q$$  
K가 Q보다 크므로, 이 반응은 정반응 방향으로 일어난다.

이제, ICE를 작성한다.

|     | $$CO_2(g)$$ | $$H_2(g)$$ | $$CO(g)$$ | $$H_2O(g)$$ |
| --- | ----------- | ---------- | --------- | ----------- |
| I   | 1           | 1          | 0         | 0           |
| C   | $$-x$$      | $$-x$$     | $$+x$$    | $$+x$$      |
| E   | $$1-x$$     | $$1-x$$    | $$1+x$$   | $$1+x$$     |

이제 이걸로 다시 $$K$$를 계산해준다.

$$\frac{(1-x)^2}{(1+x)^2} = 0.64, x=0.4444...$$

$$x = 0.44$$

이제 E만 다시 써주면 문제가 요구한 답이다.

> $$P_{CO_2(g)} = 0.56 atm$$  
> $$P_{H_2(g)} = 0.56 atm$$  
> $$P_{CO(g)} = 0.44 atm$$  
> $$P_{H_2O(g)} = 0.44 atm$$

# 4. Effect of changes in conditions on an equilibrium System

## 1. 르샤틀리에 원리

평형에 있는 어떤 계에 변화가 가해지면 그 변화를 감소시키려는 방향으로 평형의 우치가 이동한다.

기체 상태의 반응물이나 생성물을 첨가하거나 제거하는 경우 평형상태에서

- 반응물을 첨가하거나 생성물을 제거하면 정반응이 진행된다.
- 생성물을 첨가하거나 반응물을 제거하면 역반응이 진행된다.
- 고체의 첨가는 반응 지수에 영향을 주지 않으므로 평형을 이동시키지 않는다.

예시로,
$$N_2(g) + 3H_2(g) \rightleftarrows 2NH_3(g)$$에서,  
$$N_2$$ 나 $$H_2$$를 첨가하면 정반응 쪽으로 반응이 진행되고,  
$$NH_3$$를 첨가하면 역반응 쪽으로 반응이 진행된다.

## 2. 압력과 평형 이동

기체 성분을 포함하는 반응계의 압력을 변화시키는 방법

- 용기의 부피 변화
- 비활성 기체의 첨가

용기의 부피변화

- 용기의 부피를 감소시켜 압력을 증가시키면 기체의 몰수가 감소하는 쪽으로 평형이 이동한다.
- 용기의 부피를 증가시켜 압력을 감소시키면 기체의 몰수가 증가하는 쪽으로 평형이 이동한다.

압력변화

- 압력을 증가시키면 기체의 몰수가 감소하는 쪽으로 평형 이동
- 압력을 감소시키면 기체의 몰수가 증가하는 쪽으로 평형 이동

비활성 기체의 첨가

- 일정한 부피의 용기에 첨가했다면, 비활성 기체의 분압만 증가하기 때문에 평형은 이동하지 않는다.
- 일정한 압력의 용기에 처가했다면, 반응 물질이나 생성물질의 기체 분압이 감소하기 때문에 평형은 기체의 몰수가 증가하는 방향으로 이동한다.

온도와 평형 이동  
평형 상태에서

- 반응계의 온도를 높이면 흡열 반응 쪽으로 평형이 이동한다.
- 반응계의 온도를 낮추면 발열 반응 쪽으로 평형이 이동한다.

## 3. 평형 상수(K)의 온도 의존성

$$ln(K) = - \frac{\Delta H^\circ}{R} \frac{1}{T} + \frac{\Delta S^\circ}{R}$$

- van't Hoff 식

$$ln(K) = - \frac{\Delta H^\circ}{R}[\frac{1}{T_2} - \frac{1}{T_1}]$$

## 4. 반응 수득률의 극대화

수득률:

- 이론 수득량: 주어진 반응물에서 얻을 수 있는 생성물의 최대한의 양
- 백분 수득률: $$\frac{실제수득량}{이론수득량} \times 100%$$

수득률에 영향을 끼치는 요인

1. 온도에 의한 영향

- 발열 반응의 경우: 온도를 낮추면 생성 물질의 수득률 증가
- 흡열 반응의 경우: 온도를 높이면 생성 물질의 수득률 증가

2. 압력에 의한 영향

- 분자 수가 증가하는 기체 반응의 경우: 압력을 감소시키면 생성 물질의 수득률 증가
- 분자 수가 감소하는 기체 반응의 경우: 압력을 ㅈ으가시키면 생성 물질의 수득률 증가

하버법에 의한 암모니아의 합성

$$N_2(g) + 2H_2(g) \rightleftarrows 2NH_3(g), \Delta H = -92kJ$$

- 암모니아의 수득률을 높이기 위해서는 압력을 높이고, 온도를 낮춰야 한다.

(온도를 너무 낮추면 반응속도가 느려진다.)
